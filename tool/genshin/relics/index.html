使用说明

选择角色和等级 ( 80突破以下直接使用80突破即可 )

然后选择好时之沙、空之杯和理之冠的 主词条

打开 圣遗物详情 面板，在表格中填入对应的数值

填写武器攻击力白值 ( 即面板中武器的基础攻击力数值 )

数值解读

对于一般的 C，普遍认为 20~25条 有效词条属于 小毕业区间，25~30条 属于 毕业区间，胡桃等特殊角色可能需要的有效词条数量会更多

但是数量越多，就意味着毕业程度越高吗？未必

副词条数量则是一个总的加法结果，但我们都知道伤害是由各种区间组成，同样的数值上限，如果能将词条合理地分配到不同的区间里，通过乘法可得知，总的伤害会更高

打个比方，圣遗物副词条容易歪攻击，但是 攻击 是这个游戏最容易稀释的区间：宗室4、千岩4、讨龙等等

相比之下，暴击、爆伤 这些很难靠角色拐出来的区间就珍贵的多

因此，副词条总数只能是毕业与否的一个参考标准，而不是绝对标准

但我们依然可以通过参考各个区间的副词条数量，可以清晰地看出来你的一套圣遗物里副词条的进步空间在哪
